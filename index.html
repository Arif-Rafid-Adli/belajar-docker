Sistem ini berjalan 100% di dalam Container Nginx

🌐 Jaringan Docker (Network Engineer Tools)

docker network ls

Melihat semua virtual switch/vlan yang ada di Docker.

docker network inspect [nama]

Melihat detail IP Allocation dan host yang terhubung ke jaringan.

docker exec -it [container] ping [nama-container-lain]

Menguji DNS resolution internal Docker antar-container.

🛠️ Perintah Dasar Manajemen

docker ps -a

Melihat status semua container (Up/Down/Exited).

docker logs -f [nama-container]

Melihat log trafik atau error secara real-time (seperti 'terminal monitor' di Cisco).

docker compose down

Menghapus seluruh topologi lab tanpa menyisakan sampah konfigurasi.

🔍 Checklist Troubleshooting (Golden Rules)

  • Halaman Welcome Nginx Muncul? Berarti volume mounting file nginx.conf salah arah atau tidak terbaca.
  • Error Port Already Allocated? Port di laptop Anda (misal 8080 atau 80) sedang dipakai aplikasi lain. Ganti angka di sebelah kiri parameter port ("8081:80").
  • Container Tidak Bisa Saling Ping? Pastikan kedua container berada di bawah blok networks: yang sama di Docker Compose.
